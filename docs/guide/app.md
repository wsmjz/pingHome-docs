# 微架构
 
工程目录下`src`是主应用的源码，`apps`是子应用目录，每个应用单独一个文件夹存放。 子应用创建命令参考
- 应用通信
   - web components(采用)

## 主应用

当子应用创建后，需要在主应用注册，才可以通过路由前缀启动对应的子应用。
- 主应用为vue2.0
- 内置vue3.0子项目
   - 拖拽树插件
   - css交互效果
   - 高级动效
- 内置react子项目
